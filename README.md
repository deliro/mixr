# mixr

[English version](README_en.md)

Утилита для записи случайной музыки на флешку.

mixr сканирует указанную папку, случайным образом выбирает аудиофайлы в пределах заданного размера и копирует их на целевой носитель. Идеально подходит для заполнения флешки музыкой в машину.

## Установка

```bash
cargo install --path .
```

## Использование

### TUI режим (интерактивный)

Запустите без аргументов:

```bash
mixr
```

Откроется интерактивный интерфейс с полями для ввода параметров, автодополнением путей и выбором дисков через `Ctrl+D`.

### CLI режим

Укажите источник и назначение:

```bash
mixr ~/Music /Volumes/USB
```

### Примеры

```bash
# Заполнить флешку музыкой из ~/Music
mixr ~/Music /Volumes/USB

# Ограничить размер до 4 ГБ, только mp3 и flac
mixr ~/Music /Volumes/USB --size 4G --include mp3,flac

# Исключить live-записи и файлы меньше 1 МБ
mixr ~/Music /Volumes/USB --no-live --min-size 1M

# Исключить форматы wav и wma
mixr ~/Music /Volumes/USB --exclude wav,wma

# Сохранить оригинальные имена файлов
mixr ~/Music /Volumes/USB --keep-names
```

## Функции

- Случайный выбор файлов в пределах доступного места
- Фильтр по расширениям (`--include`, `--exclude`)
- Фильтр live-записей (`--no-live`)
- Минимальный размер файла (`--min-size`)
- Ограничение общего размера (`--size`)
- Сохранение оригинальных имён (`--keep-names`)
- Прогресс копирования с индикатором скорости и ETA
- Автодополнение путей в TUI
- Быстрый выбор дисков через `Ctrl+D`

## Переменные окружения

- `MIXR_LANG` - язык интерфейса (`en` или `ru`). По умолчанию определяется автоматически по системной локали.
